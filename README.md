# ADGuard
Esse repositório contem as listas de bloqueio/desbloqueio.

Para baixar o container que já está configurado [clique aqui](https://github.com/weducation/ADGuardContainer).

Para acessar a Wiki e ver como configurar, [clique aqui](https://github.com/weducation/ADGuard/wiki)

Este arquivo deverá ser importado pelo link abaixo:

    https://raw.githubusercontent.com/weducation/ADGuard/main/SitesPermitidos.txt
